# Apportable

> iOS porting

Apportable allows developers to cross-compile Objective-C apps to Android, without maintaining a separate code base in Java. It is used by many top-rated iOS games like Kingdom Rush and Osmos to deploy to Android stores using their existing Objective-C or C++ code.
