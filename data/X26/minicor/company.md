# Minicor

> Run self-healing desktop automations at scale

Minicor is a platform for building and running desktop automations at scale. One API call triggers a full desktop workflow on a Windows VM. On-premise, cloud, or Citrix. Deploy your automations with self-healing, video replay, and horizontal scaling built in. 

Go live with any customer using desktop software at scale.
