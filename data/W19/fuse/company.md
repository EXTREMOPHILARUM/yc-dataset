# Fuse

> Unified API for financial data aggregators.

Fuse provides single API that integrates with multiple bank connection vendors, such as Plaid, MX, Teller, Finicity and Yodlee. Users can easily search for and select their bank via a drop-in component in your app, and then the most reliable bank connection vendor for that bank will be displayed. Fuse also provides a consistent data model for all bank integrations and handles the necessary data transformations and API quirks for you.

By using Fuse, companies can improve the reliability of bank connections, reduce disconnections, and support more banks.
