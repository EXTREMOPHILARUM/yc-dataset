# Dedalus Labs

> Connect any model to any tools and ship hosted agents in minutes.

1. Multi-model Agents SDK: build a productionizable tool-calling agent in 5 lines of code using our open-source, vendor-agnostic SDK. Connect multiple models across providers (eg. Opus 4.6 + Gemini 3 Flash + GPT 4o) within the same agent to optimize your agentic workflow's cost, latency, efficiency, and performance.

2. Agents-as-a-Service (AaaS): serve your agent or service as a hosted endpoint that anyone - including OpenClaw and Claude Code - can call.
