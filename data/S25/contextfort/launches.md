# Launches

## ContextFort - Copilot for Construction Drawing Review

# **tl;dr**

ContextFort automatically finds missing dimensions and overlay mismatches across your architectural and structural drawings and drafts Requests For Information (RFIs). Watch our [demo](https://www.youtube.com/watch?v=2C4Gfe4UoJA) here.

# **The Problem**

Contractors plan their construction using drawings prepared by the design team (Architects, Structural Engineers). But before building, they need to review them for errors and ask for clarification by submitting a **Request For Information (RFI)**.

Today, this review is done by **manual pageturn and eyeballing overlays** which takes up **weeks**. And, this has to be done every time a modified version is released.

If an inconsistency goes unidentified or an RFI is not sent early, the project gets **delayed** or ends up needing **costly rework** with change-orders having uncertain payments.

**The problem goes beyond this.** Corrections following these RFIs affect multiple trades but they are never notified; for example, a concrete contractor moving a column by 3 inches can affect HVAC and elevator contractors. This poor cross-trade visibility has been the cause for a majority of reworks.

# **Our Solution**

Today, ContextFort saves contractors **2-3 weeks/project** by using AI to:

1. Identify missing and inconsistent dimensions across architectural and structural drawings
2. Find overlay mismatches between different versions.
3. Ensure RFIs being generated are not already in the construction documents.
4. To draft an RFI following the format laid out in the spec sheets.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=92604&key=user_uploads/1596702/7b5f2e9e-d0d6-4ad6-8bd2-2e09171ba7b6)

Our vision is to turn every drawing set into a **living, shared model:**

1. One that tests itself with rules that capture every trade clearances, codes, scheduling constraints and run automatically on every edit
2. Only the teams affected by a change get pinged– no more hunting through 200 sheets to see what moved
3. Each contractor views a filtered lens of the model that shows precisely what they must build and how other trades’ moves impact them

**Our mission is to turn RFIs into a rare exception and have projects finish first-time-right.**
