# KERNEL

> Crazy fast, open source infra for AI agents to use the Internet

Kernel is crazy fast, open source infra for AI agents to access the internet 🧠🕸️
- Built on unikernels for crazy fast cold starts and low resource usage
- Sub-150ms cold starts on managed headful browsers
- Built-in stealth mode with residential proxies and automatic CAPTCHA solving
- Full observability: session recording, live view, and full logs
- Persistent sessions that maintain cookies and auth across invocations
- Native support for all browser and computer use frameworks
- SOC 2 compliant and HIPAA-ready

Trusted by Cash App, Rye, and 1000+ others. Backed by Accel & Y Combinator (S25).
