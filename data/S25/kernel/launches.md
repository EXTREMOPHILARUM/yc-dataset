# **🤝 TLDR:**

We make an API to instantly launch browsers in the cloud so AI can use the internet just like humans do.

# **🚀 What’s Kernel?**

Kernel is the best place in the world to launch web agents and browser automations. We handle the hard parts of running browser infrastructure in the cloud so you don’t have to: spinning up browsers quickly, responsive read/write live view, logs, headless vs. headful mode, residential proxies, and more.

Our edge is an infrastructure platform that’s **flexible, observable, and insanely fast** from day one:

* **Performance** - web automations and Computer Use agents can launch and connect to Kernel browsers in 300 milliseconds
* **Isolation x telemetry** - Kernel browsers run in isolated virtual machines and ship with live views & replays so you can see exactly what your agent saw and take control when needed
* **Agent runtime** - launch your web agent on Kernel’s code execution platform to seamlessly connect it to a browser in the cloud
* **Simple, clear pricing** - pay only for the resources you use. Our straightforward pricing model makes it easy to predict usage costs

# **⚡️ How does it work?**

Kernel’s API makes it simple to launch browsers in the cloud and automate anything on the internet. **We work with any browser/web agent framework**: **Playwright, Puppeteer, Browser Use, Magnitude, and more.** Users love our platform for:

* Instant **browser spin-ups** on-demand
* **VM logs & read/write live view**
* Video **replays**
* **Browser session** reuse
* **Residential proxies** and auto-CAPTCHA solvers
* **Turn any agent into a https endpoint** — we host your agent, you invoke it on-demand with a prompt
* & much more soon!
