# Launches

## Nottelabs: Web Agents for Enterprise

**TLDR;** Notte provides a platform to build and deploy web AI agents that don’t break. We turn messy, repetitive web tasks into reliable, self-healing workflows so businesses can scale automation with confidence.

https://www.youtube.com/watch?v=uwTOpQTUfyk

**Context**

We built Notte because the web was made for humans, not AI agents. Our first product was a perception layer that helps LLMs parse and understand site structure, boosting agents’ performance. But 90% isn’t enough to scale enterprise workflows with confidence.

**Agent Workflows**

That’s why we launched **Agent Workflows** - where web automation meets agentic reasoning. Start an agent to discover the workflow, freeze the routine into deterministic steps, and keep agents only where it’s needed to unlock new capabilities. If a site changes, agents dynamically close the gap to ensure workflows never break.

Less tokens = Less expensive = More control = Faster automation

**Full-stack Ecosystem**

* **Hosted Sessions** - Stealth mode and auto-scaling browser
* **Vaults** - Credentials, MFA tokens, SSO, and cookies securely
* **Personas** - Emails and phone numbers for account creations
* **Payments** - Credit card payments and agentic transactions
* **Misc** - File IO, Replays, CDP compatibility, etc.
