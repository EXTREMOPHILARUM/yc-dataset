# Manufact (formerly mcp-use)

> Open source SDK & cloud infrastructure for MCP Servers & ChatGPT Apps 

Manufact provides open-source dev tools and infrastructure for MCP to help dev teams quickly build and deploy MCP Servers for AI agents & MCP Apps for ChatGPT and Claude.
We developed the mcp-use SDK, which has over 5M downloads and 9,000 GitHub stars, and is used by 20% of the US 500.
Create MCP servers/apps in minutes, connect your GitHub repo, and make them immediately available to AI agents and ChatGPT.
