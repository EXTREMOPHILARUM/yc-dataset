# OddsView

> The Bloomberg Terminal of Sports Betting

OddsView is the one-stop-shop for sports betting tools and media. 

The OddsView terminal allows users to browse millions of odds updated in real time to find the best bets. Users can access familiar features like positive expected value and arbitrage finders, as well as exclusive tools such as historical odds charts, all within a single platform.
