# renderlet

> Build interactive applications that run anywhere

renderlet is a framework that makes it easy to develop interactive applications.

Building applications with high-performance graphics is very hard – existing tools are cumbersome and aren’t truly platform agnostic, leading to frequent rework. 

With renderlet, developers can write high-level code to describe 2D and 3D graphics without having to use low-level, platform-specific APIs.

renderlet automatically compiles graphics code to WebAssembly and embeds a portable rendering engine inside any app, enabling graphics to safely and automatically render on any platform’s GPU.

Our runtime is open-source: https://github.com/renderlet/wander
