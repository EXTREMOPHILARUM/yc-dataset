# Ubicloud

> Open source alternative to AWS

Ubicloud is an open source cloud that can run anywhere.

Our cloud services include elastic compute, block storage, virtual networking, managed Postgres, K8s, AI inference, and powerful IAM. Ubicloud provides these services on bare metal providers, such as Hetzner, Leaseweb, or AWS Bare Metal.

You can self-host our software or use our managed service to reduce your cloud costs by 3-10x.
