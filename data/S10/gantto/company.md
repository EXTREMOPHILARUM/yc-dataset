# Gantto

> Better Microsoft Project.

Gantto was conceived to fill one obvious gap in our weekly management routine. Each week we would collect status information based upon what was really happening inside the organization and then update our project models to reflect and digest the changes. Then we would spend tedious hours manually creating presentations based upon the same underlying project data that we had already updated. After a long search for a good solution we set out to create our own.
