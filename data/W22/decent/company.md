# Decent

> One-click transactions using any token across chains.

Decent's core product,[The Box](https://decent.xyz/), enables 1-click transactions using any token across chains.  For example, a user can purchase an Optimism NFT off primary or secondary sales using USDC on Arbitrum.

As applications splinter across an increasing number of rollups and app chains, wallets and dapps will have to meet users at their point of liquidity vs. expecting users to jump through hoops to arrive with tokens on the correct network.
