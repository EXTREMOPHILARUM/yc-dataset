# Launches

## Vizzly ⚡️ Customer-Facing Analytics for Modern SaaS

**✨ TL;DR**\
[Vizzly](https://www.vizzly.co/) is customer-facing analytics for modern SaaS.

Our primary goal is to help SaaS companies build beautiful customer-facing analytics without compromising the user experience or product quality.

Vizzly offers a semantic layer, works with your authentication setup, and provides a powerful visualization layer that's extensible with code for those who want more control.

With Vizzly, customers can build and deliver anything from simple user-facing insights to advanced custom reporting and everything in between.

**❌ The problem**

Over 75% of B2B SaaS companies have (or will need) customer-facing dashboards or reporting capabilities, but embedded analytics tools are stereotypically rigid and force a lot of compromise.

For example, many of these tools offer simple iFrame embeds, instantly limiting flexibility, programmability, and impacting performance.

If SaaS companies do build in-house, they end up committing precious time to building a large product feature that’s often not core to their unique value proposition.

The maintenance overhead is often overlooked until customers begin asking for new cuts of data or other views, meaning the SaaS company needs to extend the backend API and build new visualizations if they want to satisfy the request.

**💡 How we help**

Vizzly provides customer-facing analytics for modern SaaS; offering the same level of flexibility as an in-house development process with minimal technical overhead required.

Key product areas include:

* Ultra low-code experience: get setup in hours.
* JS-based embedding (React, Angular, Vue & Svelte).
* Extensible with code and highly flexible.
* Native to your UI (we offer a lot of control).
* Works with normalized and dynamic customer data structures.
* Direct API access via our React or Vanilla JS Hook.
* A powerful Custom Reporting option.
