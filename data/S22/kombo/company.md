# Kombo

> Unified API for HR, payroll and recruiting software

Kombo makes it easy for software companies to offer integrations with HR software. For example, a recruiting company could use our unified API to integrate with the dozens of recruiting systems used by different customers all at once instead of building an integration to each of them.
