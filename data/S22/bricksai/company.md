# BricksAI

> Streamlining LLM application management for enterprises

BricksAI is an open-source AI gateway that helps you manage how each of your services is calling different LLM providers.

You can create API keys with rate-limits, spend-limits, and expiration dates, then use them to make requests to different LLMs. With our SSO dashboard, you also get (1) role-based access control, and (2) fine-grained metrics (e.g. cost, number of requests, tokens) over each API key.
