# winfunc

> ai-native security engineering for mission critical codebases

Winfunc (winfunc.com) is an AI Hacker that autonomously finds, verifies, and patches security vulnerabilities in codebases, just like a human security engineer would. 

Winfunc can find business logic errors with context-aware scanning and automate the full auditing cycle to generate reports with zero false positives.

Winfunc's agent has found vulnerabilities in Google, Supabase, Sentry, Cal, Gumroad, Bun, Hoppscotch, etc.
