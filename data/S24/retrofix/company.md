# RetroFix

> Automate any workflow using plain english

RetroFix is an automation platform designed to be extremely intuitive to use, making it much easier and faster to automate manual work. 

To create or modify workflows you can describe them in natural language and our agent will create them end-to-end for you. We allow you to connect with 2000+ applications and use our pre-built actions for your workflows. In the case we don't have a pre-built step, our agent can instead write code to access the full APIs of all the apps we integrate with.
