# Briefer

> The collaborative data platform.

Briefer helps data scientists and analysts build interactive visualizations and data apps using a Notion-like interface.

Connect to your data sources, write SQL or Python, collaborate through comments and multiplayer editing, and run code in whichever compute environments you need.
