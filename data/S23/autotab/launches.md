### Autotab is built for real work

“We were promised digital robots, and all we got was tab complete.”

So far, AI agents have not been reliable enough to do meaningful work without supervision.

Autotab is the first system for building extremely reliable agents that can execute any repetitive task fully autonomously. Show it how to perform your task, train it to be hallucination-proof by adding examples and providing feedback, and then run it on demand or on a schedule—either locally or in the cloud.

### Hallucination Proof

Real work is hard. To push the frontiers of autonomy, we have developed multiple systems that work together to achieve the improbable.

1. **Digital Robot**: For AI to do real work, it needs to be able to use a computer. Autotab is driven by more than 16 unique models, from fast on-device models to vision-based robotics and the largest frontier models in the world.
2. **Demonstration Learning**: Telling an AI to book flights for you is a cool demo, but real work is nuanced and difficult to explain in a prompt. You teach Autotab by _showing_ it how to do your task the same way you would a human colleague. This means Autotab has 100-1000X more data than prior approaches, providing the deep context required for full autonomy.
3. **Action Scaffold**: Each task that Autotab learns is composed of a series of steps and sub-goals. This makes it easy to understand what Autotab is doing and gives you fine-grained control.
4. **Memory**: As it runs, Autotab prompts you for feedback and provides opportunities to steer its behavior. These learnings are accumulated into action memory—improving Autotab’s world model and allowing it to work autonomously with extremely high reliability for hours on end.
5. **Bare Metal Control**: Autotab uses a custom browser for secure, bare metal task execution—locally and in the cloud. This also lets us avoid most bot detection.

The result is an AI system that learns and works like a human colleague. It uses a browser, so it can handle any repetitive task you can—there is no list of integrations, just teach it to use whichever apps you want. It asks for clarification when it is uncertain, and you can correct it if it gets something wrong.

## ZTool - Create data tools & scripts with just a few words

**tl;dr**

**Describe** the tool you need **in** **one sentence** and ZTool will create the code, run it, and present the output. We particularly focus on all things **CSV** **files**: anything from exporting csvs, to merging files and enriching data is possible. Over time, users can edit, build out and share their library of tools with each other.

**Problem**

Python scripts for tools like scrapers or workflow automations have been written thousands of times before. Reusing that code — and saving time — should be easy but it’s not: finding the relevant scripts is hard, and adapting it to your specific case requires a certain level of skill at which one can just write the code from scratch. ZTool solves both the search problem and the skill problem; it will give you the exact script you need, specific to your use case, and regardless of your technical ability.

**Solution**

All the user has to do is describe the script they want, for example: merge two CSV files. ZTool, powered by LLMs, will ask specifying questions, ask for the input files, and then create an output file. That’s it!

**How we came up with this**

While working as an analyst at a VC firm many years ago, Jonas hacked together a script to process csv data for a deal analysis. It was a simple script but it **saved him days worth of manual work**. Over time the initial script proved useful for more and more deals and grew into an entire library of Python scripts for automating different tasks, from data analysis to web scraping. It took Jonas a week to create that initial script and he had to learn Python, regex, SQL, Docker, cloud, and more—with ZTool anyone could have done it in 5 minutes.
