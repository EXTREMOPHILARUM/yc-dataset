# Skyvern

> Open Source AI Agent to automate browser workflows via an API

Skyvern helps companies automate browser based workflows using LLMs and Computer Vision, fully automating manual workflows and replacing brittle or unreliable scripts.

Examples include:
1. Automating materials procurement from commerce websites
2. Completing complex multi-step workflows (ie getting an insurance quote from Geico.com)
3. Automatically logging into portals and downloading invoices
4. Navigating legacy content systems to do data extraction or data entry
