# Flowise 3.0

https://youtu.be/-h4WQuzRHhI

For the first time, anyone can build AI agents from just an idea. Simply describe what you want to create, and a draft will be generated for you.

Flowise provides modular building blocks for you to build any agentic systems, from simple compositional workflows to autonomous agents. Some common patterns are:

* Chaining
* Routing
* Parallelization
* Hierarchy
* Looping
* Iteration

![uploaded image](https://www.ycombinator.com/media/?type=post&id=90360&key=user_uploads/1358352/68095851-704f-4f14-b1cf-dfcfc56b4009)

Each LLM/Agent has the following augmentations to enable flexible multi agents architecture:

* Memory
* Retrieval(RAG)
* Tool
* Centralized State
* Structured Output

🙋‍♂️ **Human in the Loop**:

This is a core aspect of an agentic system, allowing humans to review tasks performed by agents, or to approve/reject tool calls by agents. For example, reviewing actions before sending an email or booking an appointment.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=90360&key=user_uploads/1358352/a08091a2-93cb-493f-858e-a851bcd3d12d)

📝 **Form Input**:

In addition to the standard chat interface, workflows can also be triggered through a form-based interface. This is ideal for workflows that don't require back-and-forth communication between the user and agents.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=90360&key=user_uploads/1358352/bba6f3b4-3d95-4699-bcdc-b6c2485f1fa2)

🔍 **Observability**:

Gain full visibility into execution traces to debug issues and optimize performance. You can share traces publicly, allowing others to review, approve, or provide feedback.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=90360&key=user_uploads/1358352/4e8ed542-c2f8-424c-8ef2-0c60c059199a)

✅ **Flow Validation**:

Designing complex workflows can be challenging. Flow validation helps minimize misconfigurations and human errors by automatically checking for common issues.

## Flowise - Low-code LLM apps builder

### **☕️TL;DR**

[**Flowise**](https://flowiseai.com/) **is an open source drag & drop tool to build your customized LLM flow.**

**We provide a visual interface to let you build backends for LLM apps used for QnA, summarization and analysis on your own data.**

![uploaded image](https://www.ycombinator.com/media/?type=post&id=72776&key=user_uploads/1358352/f29f7b92-9f41-4257-b293-046eb46d9b84)

👋 This is [Henry](https://www.linkedin.com/in/zhenjing-heng-henry/) and [Yau](https://www.linkedin.com/in/chung-yau-ong-1407701a9/), co-founders of Flowise. We’re on a mission to make it easy for people with no-code background to learn and build AI apps.

### **🤔 Problem**

Recent advancement of large language models like ChatGPT is revolutionizing every industry. But how do people from less tech-savvy background be able to learn it, and use it for their own use cases?

### **✨ Solution**

Just like LEGO[🧩](https://emojipedia.org/puzzle-piece/), Flowise let you connect different components like PDF loader, OpenAI Embeddings, Pinecone vector database and build your own ChatGPT on your own documents. You learn as you build!

### **⚙️ How it works**

**Chatflow**

Drag & drop any component to the canvas and interact with it in real-time.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=72776&key=user_uploads/1358352/064c1f3e-83c2-489f-a115-b7efdd74289f)

**API/Embed**

Use the flow as an API or embed into your application.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=72776&key=user_uploads/1358352/4c5d5d78-eedd-48b0-91cf-e0c7fa1bac53)

**Tool**

Create your own tool and let agent automatically use it. For instance, get stock prices and add it to Airtable/Gsheets.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=72776&key=user_uploads/1358352/558cf99b-b199-48d9-a0fa-d1500a30ec7d)

**Extensible**

Flowise is built to be extensible. Developers can easily customize their integrations by using libraries/frameworks such as LangChain, LlamaIndex, HuggingFace, etc.
