# Inconvo

> Build chat-with-data agents for customer-facing applications

Inconvo is the open-source platform for building data agents on production data. With safe queries, permissions, and structured outputs.

Use Inconvo to build and deploy data agents that query production databases and can be called directly from your application. You get safe query execution, permissions, stateful conversations, and observability out of the box.
