# Launches

## Sponge: Financial infrastructure for the agent economy

**TLDR**: 

Sponge gives agents access to bank accounts, cards, and crypto so they can operate as **independent economic actors**. Sponge enables businesses to sell directly to agents **without human involvement**.

<https://www.youtube.com/watch?v=9Db70ieMYkA>

**The Problem:**

Long-running agents are starting to handle complex tasks entirely on their own. They can plan, execute, and complete multi-step workflows without human intervention. But that autonomy ends the moment a payment is involved, because transactions still require a human to manually enter payment details and authorize the charge.

**The Solution:**

To solve this, Sponge built infrastructure that lets agents handle payments on their own.

**Sponge Wallet** enables agents to sign up and pay for services without human involvement. Users fund their wallet once, and agents autonomously use those funds to pay for what it needs.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=97911&key=user_uploads/300418/a9904dc0-f667-4060-9697-e78f4a955155)

**Sponge Gateway** enables businesses to sell directly to agents. Businesses can upload their API spec and allow agents to onboard and pay for usage autonomously, without requiring a human to give a credit card or an API key. No code changes required.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=97911&key=user_uploads/300418/0c13e43e-b0b7-4400-9c8c-79408aa9eaba)

As agents become increasingly autonomous and begin making decisions on their own, they need financial primitives to operate like humans. We’re building the infrastructure that makes this possible.

**Our Ask:**

If you use agents (such as OpenClaw), add our [wallet skill](https://sponge.md) to let them sign up and pay for services autonomously. If you're a business, upload your API spec to [gateway.paysponge.com](http://gateway.paysponge.com) to start onboarding agents and selling to them directly, without human involvement.

**The Team:**

We're a team of ex-Stripe tech leads that built Stripe's stablecoin financial accounts, stablecoin payments and core money movement systems for humans. We are now bringing those learnings to build out the financial infrastructure for the next economic actors: agents.
