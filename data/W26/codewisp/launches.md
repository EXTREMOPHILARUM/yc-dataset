# Launches

## CodeWisp - AI Game Creation Platform

Hey everyone, I’m [Elvin Fu](https://www.linkedin.com/in/fuelvin/) - founder of [CodeWisp](http://codewisp.ai).

**TL;DR**

**CodeWisp lets anyone create and publish web games using prompts.** Describe the game you want and CodeWisp generates it - no need to juggle code files, engines, or asset pipelines.

<https://youtu.be/u9YROPUUHSk>

**Why this matters:**

Gaming is already a massive industry - it’s expected to grow to nearly **$300B globally by 2029** _(PwC Global Entertainment & Media Outlook)_.

But the tools used to build games are still designed for professional studios.

AI is creating a new paradigm where games can be **generated, modified, and shared instantly,** opening game creation to millions more people.

That is the opportunity we’re building toward at CodeWisp.

**The Goal:**

Remove the friction of traditional game development and make creating and publishing games accessible to anyone, not just engineers or studios.

**The Product:**

CodeWisp has two main parts (everything is in browser, no downloads or tools required):

1 ) The Editor\
A browser-based workspace where you describe the game you want and CodeWisp generates it. You can prompt edits to add mechanics, enemies, physics, levels, or visuals and see the game update instantly.

CodeWisp can generate 2D games, 3D games, and multiplayer browser games.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=98433&key=user_uploads/892590/a884349c-6658-45ae-b86d-65a6cf2c9129)

2 ) The Homepage\
A place to explore and play games created by the community. Every game can be opened, played in the browser, and shared instantly with a link.
