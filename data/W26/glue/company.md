# Glue

> The Interface Design Canvas for AI agents

Glue is an open source design editor for AI agents. It gives coding agents an infinite canvas to create new interfaces, iterate on existing ones, and turn those changes into production-ready code all in real time.

The rise of coding agents is collapsing EPD into one role: builders. This new paradigm means design and engineering can now converge to a single workflow. Builders no longer want static mockups in one tool, and implementation in another.

Glue is a collaborative workspace where builders and agents can create new components, iterate on existing interfaces, and implement as code. The same tight feedback loop that once made Figma powerful for designers is now built for builders and their armies of agents. Being open source allows us to penetrate workflows at zero cost and build trust quickly.

Right now there are no design tools with AI as the primary user. Existing tools bolt on copilots as an afterthought. Soon, every company will have AI agents building their interfaces. We are building the surface for this workflow.
