# Scaphold.io

> The Universal API

Scaphold is the world's most powerful GraphQL as a service platform. The platform will provide a number of services including push notifications, analytics, file storage, custom code, multiple environments, and more to allow developers to build mobile, web, and IOT applications. Our innovative use of GraphQL is the big game changer here. We’re convinced that it’s the future of APIs and we want to make it readily accessible to everyone else.
