**We turn natural language into lab automation.** Scientists type their experiment in English. Our software programs robots to execute it.\

# **Problem**

The biggest scientific breakthroughs -- _think Selman Waksman mining microbes for antibiotics or Akira Yoshino testing early lithium-ion batteries_ _at Sony_ -- started in labs through painful, manual work. **Decades later, discovery still relies on pipetting, weighing, and prepping by hand.**

It’s inefficient, exhausting, and slows down discovery. Automation solutions exist, but are often **$$$** or built for **specific use cases**. 

We want to change that.

# **Our Take**

What if you could run an experiment by just typing it out? 

That’s what we’re building for: **automation that’s usable by everyday scientists.**

# **The Zeon System 🦾**

We sell a hardware + software system that runs lab experiments using off-the-shelf robotics. **Labs receive a ready-to-run automation setup** that includes:

* Our AI-powered software
* 2 off-the-shelf robotics arms with mounted depth cameras

_How it works:_

* Scientists describe protocols in plain English
* Zeon interprets them, detects lab tools using computer vision, and plans the execution
* The system generates and executes robotic code in real-time, adapting to the specific lab environment

# **What’s Possible**

We are already piloting with labs at UCSF and Stanford to:

* Run fluorescence screens for nanoparticles overnight
* Automate multichannel pipetting for clinical assays
* Dispose of biohazardous waste while researchers sleep

# **Big Picture**

Eventually, we want to **enable labs that run themselves**. Think self-driving labs on Mars or in the middle of the ocean doing science while humans dream. 🪐 🧪🧬
