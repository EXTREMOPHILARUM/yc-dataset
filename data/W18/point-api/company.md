# Point API

> Point stores things you send over and over again and helps you re-use…

Point stores things you send over and over again and helps you re-use them instantly.

Point can store ANYTHING, including long email templates, links, images, and emojis! When you want to re-send your pitch, support ticket resolution, or even just a Calendly link, Point makes it easy to find and reuse it so you never have to type it out again.

No need to search for your favorite emoticon anymore...¯\_(ツ)_/¯

Install our Chrome Extension to use Point anywhere on the web: https://pointapi.com/chrome.
