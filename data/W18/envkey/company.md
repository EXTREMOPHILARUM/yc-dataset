# EnvKey

> End-to-end encrypted configuration and secrets management

EnvKey is a developer-friendly, cross-platform, end-to-end encrypted configuration and secrets manager. It can be used to store API keys, database credentials, and other application config securely. 

EnvKey makes it easy to grant least-privilege access to the developers and servers that need it from a user-friendly UI. Integration takes minutes.

Using EnvKey ensures that no 3rd party, including EnvKey itself, ever has access to your secrets. 

It takes a part of your system that used to be error-prone and insecure and makes it just work.
