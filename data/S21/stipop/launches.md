# Launches

## Stipop: A must-have sticker feature in modern communication

Hi everyone [👋](https://emojipedia.org/waving-hand/#:\~:text=A%20hand%20waving%20most%20commonly,to%20Emoji%201.0%20in%202015.) [Tony](https://www.linkedin.com/in/tonypark-stipop/) and [Daniel](https://www.linkedin.com/in/danielcho6/) here to introduce [Stipop](https://stipop.io/en/)!

Stipop provides an **easy-to-install sticker API and SDK** to help you install a complete sticker feature within minutes. It includes 150K+ unique stickers designed by thousands of creators, a pre-built and customizable user interface and a sticker search engine.

### **Why stickers in modern communication?**

Online communication has become ever more important and users are always looking for ways to express their thoughts and emotions better. That’s why stickers are now used 17x more than GIFs worldwide and from experience on FB and Snapchat, users expect high-quality stickers in every online community they are in.

### **How it works**

1. Sign up on Stipop Dashboard
2. Invite developer and integrate Stipop API or SDK
3. Customize UI and sticker list from the Dashboard
4. Release and see how it affects user engagement
