# Enso

> Self-Service Data Prep and Blend Built for Data Teams.

Enso Analytics is a self-service data prep and analysis platform for data teams. It transforms complex processes into repeatable workflows, saving users 7 hours weekly. With full configuration visibility, seamless environment transitions, and real-time updates, Enso caters to both non-programmers and developers, offering cloud and desktop options.
