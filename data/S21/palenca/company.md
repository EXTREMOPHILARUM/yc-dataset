# Palenca

> Payroll API for LatAm

Palenca is Payroll API for LatAm. We make it possible for any worker to share their employment records and for businesses to uses those records. Like Plaid, but for Payroll.

Unicorns like Kavak and Lalamove use us to run background checks and provide financial services.
