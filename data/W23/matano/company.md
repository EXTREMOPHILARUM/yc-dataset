# Matano

> Cloud Native SIEM (alternative to Splunk/Elastic)

Matano is a modern SIEM, built for cloud-first security teams. 

It replaces traditional SIEM databases like Splunk or Elastic with a cybersecurity platform built on top of a cost-effective Security Data Lake.
