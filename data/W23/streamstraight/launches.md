# Launches

## Lightski: Cursor for Engineering Docs

Lightski is the **Cursor for Engineering Docs: a rich-text editor and AI assistant specifically built for engineering managers and tech leads**. Whether you’re writing a technical design doc, internal documentation, PRD, or peer review, Lightski helps you write higher quality documents faster with the help of AI.

### Engineers (broadly speaking) hate writing

As an eng manager and tech lead, I spent a significant chunk of my time writing all kinds of documents: design docs, post-mortems, internal documentation, etc. Luckily I enjoyed it, but most didn’t! We built Lightski because AI tools like Cursor have significantly sped up how engineers write code, but not yet for documents.

Whether you’re writing a technical design doc for your promo packet or summarizing an incident in a post-mortem, Lightski helps you create a polished document in less time.

### In a world of AI Slop, Lightski emphasizes Human Expertise

Lightski does not generate documentation on autopilot. That kind of documentation has little value, because the real value is in the human insight and context that does not exist outside of your brain. Instead, **Lightski assists with the act of writing**—turning your unique idea into a polished document.

We’re building the best possible text-editor for humans to communicate in a written format. As long as you provide that idea, Lightski helps with the tedious bits! For example, Lightski can quickly convert code into a Markdown table:

How can Lightski help? Lightski natively bundles AI with a powerful text-editor and access to Github. By directly working with the document, our assistant can:

* Explain the logic in your codebase
* Pull schemas from code into a document
* Rewrite code into tables or bullet points
* Create a draft with references to your codebase
* Brainstorm different approaches to a gnarly problem
* Search and reference online documentation
* Polish up wording and fix grammar
* and more!

## Lightski - Embed an AI data scientist in your app

**tl;dr:** Lightski helps you build an analytics experience for your users that answers any arbitrary question about their data. We provide embedded data analytics more powerful than Tableau and Looker by pairing AI with code execution, which eliminates hallucinations.

## Why?

Before YC, I was an engineering manager at Affinity CRM, where we spent six figures on an embedded analytics solution. Even then, our users were never happy; they either didn’t know how to navigate the complex UI or wanted even more advanced analytics. Most customers ended up exporting the data themselves and running their own analysis (not an ideal solution).

Recent AI advancements remove this trade-off, enabling a natural language interface and powerful code-based analysis. We started Lightski to quickly give your users the full power of a data scientist, without needing to know a line of code.

## User Experience

With Lightski, you’re able to embed an AI Data Scientist in your app, pre-loaded with your users’ data. Your users can then explore a dataset however they want; in response, we generate and execute code to perform analyses, such as:

* Create custom graphs, charts, and dashboards
* Perform arbitrary filters, group-bys, and aggregations
* Transform or clean data for a custom CSV export

Anything that a data scientist can accomplish in code can now be accomplished with Lightski. Best of all, your users won’t even know we’re powering the experience—you can quickly re-theme our UI with your design system.

## Under the Hood

We rely heavily on state-of-the-art LLM models to guide the analysis, but there’s a lot of other capabilities built in-house that power the experience:

* Secure and isolated per-user code execution environments
* Native connectors for the most common databases
* Code-based ETL, for flexible data pre-processing and transformation
* Support for image generation and CSV export
* Embeddable streaming UI that supports custom theming

We’re building fast and will also soon launch:

* Interactive charts and even more customizable dashboards
* Direct API access to our safe code execution environments and ETL pre-processing features
* Guided assistants, for less technically-savvy users

and much more! Let us know if you’re interested in any of these and we’ll prioritize it for you.
