# Framebase

> API for video.

Framebase is programmatic video.

Writing applications which use video in a meaningful way is extremely difficult. And then there's the maintenance.

Framebase makes video a first-class object. Instead of hiring a team to deal with transcoding configurations, device compatibility bugs, or CDN contracts, Framebase lets you add two lines of code to start treating videos as class instances.

Framebase is a pivot of TapIn.tv, which is a Y Combinator S 2012 company.
