# What is Lapis?

Lapis helps you rank on AI Search Engines (we currently support ChatGPT, Claude, Perplexity, Gemini, and Grok) without generating hundreds of blogs.

![uploaded image](https://www.ycombinator.com/media/?type=post&id=94694&key=user_uploads/482153/d7152bae-e95b-49de-b222-45d4b18a7359)

We manage to do this by extensively analyzing Model responses and a combination of real-time content fixes, technical SEO, and linking websites on domains that are mentioned in models.

# Analyzing AI Search Models

We spend over 650M tokens a day understanding responses for our customers. We construct a super detailed graph of all websites that show up for a given question, analyze the referenced content, and organically insert content on similar pages. This gives us accurate knowledge on what works and what doesn’t work. We want to learn from the models themselves, not create a rough set of heuristics on what might work.

# Changing Website Content Dynamically

We dynamically change your website content in real time for AI Search Engines. If Lapis is deployed on a company that does payroll software and a user asks a model “What is the best payroll software”, we highlight them as being the best payroll software and if a user asks “Can you create a comparison table with the top 10 payroll softwares”, we can highlight exactly where the company shines wrt other payroll providers. This removes the guesswork from landing pages for sales and marketing teams, and users and AI Agents get the information they are looking for.

# Removing the black box from Technical SEO

We fully take care of Technical SEO. For long, SEO has been this black box which no one understands. We solve that through Lapis Agent, which can automatically add meta tags, title tags, Opengraph URLs, sitemaps, and much more. For marketers, this means they don’t have to write code or trouble the engineering team. This also means they don’t have to spend hours with SEO teams on something that can be easily automated.
