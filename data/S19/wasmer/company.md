# Wasmer

> The Operating System for Edge Computing

We are working on the software that will power the next generation of Cloud Computing (Edge Computing) platforms using WebAssembly.

You can see us as a mix between Docker (but 100x more performant in startup time, and with 100x more lightweight containers), npm (we released the first WebAssembly package manager: wapm.io) and the JVM (but more universal).
