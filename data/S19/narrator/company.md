# Narrator

> The only platform that can prep data to answer EVERY question you…

Narrator is an end-to-end platform built on top of the data standard, the Activity Schema and starting at $500/mo.  Data analyst are able to build their definitions of their user journey, and use that journey to answer any question that comes up.  From there, data can be visualized in a dashboard, used to build a story like analysis, exported and more.   

The biggest values of Narrator is Speed and Cost reduction.  Small teams are able to move fast and answer questions in minutes allowing them to preform the work of very large data teams. All while Narrator is optimized to minimize compute cost of the warehouse.
